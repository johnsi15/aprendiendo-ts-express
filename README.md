# Express with TypeScript 


## Initial config tsconfig
### Install TypeScript
`npm install typescript -D`

### Added tsc script to package.json
```js
  "scripts": {
    "tsc": "tsc"
  }
```
### Init config tsc
`npm run tsc -- --init`
